# Nome do Projeto:
Repositório criado para o curso de Fundamentos de IA e Análise de Dados

## Descrição:
Este curso apresenta os fundamentos da linguagem Python e evolui progressivamente para o ecossistema de ferramentas mais utilizadas em ciência de dados. O conteúdo parte da base da linguagem (sintaxe, variáveis, estruturas de dados) e avança para bibliotecas especializadas em manipulação numérica, análise exploratória, visualização de dados e aplicação de técnicas de Inteligência Artificial para gerar insights e métricas relevantes.

## Tecnologias utilizadas
- Python 3.x
- NumPy
- Pandas
- Seaborn
- Matplotlib (suporte ao Seaborn)
- Jupyter Notebook (opcional, recomendado para as aulas práticas)
